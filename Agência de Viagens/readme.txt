# Sobre o projeto

A proposta desse projeto é criar uma página web para uma agência de viagens, utilizando as tecnologias HTML e CSS. O objetivo do projeto é fazer um página um formulário, para que o usuário preencha e envie seus dados. Além disso, o formulário (assim como o resto do site) é responsivo, adaptando-se à diferentes telas.

# Tecnologias utilizadas
##Front end
- HTML 
- CSS

##Back end
- PHP (O back end foi realizado e disponibilizado para uso pela equipe do DevMedia, não tendo participão minha no código php)

# Sobre o projeto

A proposta desse projeto é criar uma página web para uma pousada fícticia, utilizando as tecnologias HTML e CSS. A página contém tags HTML básicas, estilização via CSS com o uso de FlexBoxes, além de ser uma página responsiva quanto a adaptação para telas de celular.

# Tecnologias utilizadas
##Front end
- HTML 
- CSS

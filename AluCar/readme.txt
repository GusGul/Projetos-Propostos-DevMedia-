# AluCar - Aluguel de Carros
[![NPM](https://img.shields.io/npm/l/react)](https://github.com/devsuperior/sds1-wmazoni/blob/master/LICENSE) 

# Sobre o projeto

A proposta desse projeto é criar uma página web para uma empresa que disponibiliza o serviço de aluguel de carros, utilizando as tecnologias HTML e CSS. O objetivo do projeto é fazer uma página com um formulário, para que o usuário preencha dados de texto, número, telefone e datas, além de selecionar entre algumas opções de carros.

# Tecnologias utilizadas
##Front end
- HTML 
- CSS

##Back end
- PHP (O back end foi realizado e disponibilizado para uso pela equipe do DevMedia, não tendo participão minha no código php)
